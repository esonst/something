## robot V1.0 使用说明
运行时，产生的文件说明如下:
**dataset文件夹内保存着验证码的数据，请攒多了打包为zip发至邮箱eson.xin@gmail.com**

| 文件名        | 作用   |  
| --------   | -----:  | 
| pass.txt     | 保存用户名密码 | 
| logcode.gif        |   登陆验证码   |
| verycode.gif        |    选报告验证码   | 

本软件在使用时，刷新时间间隔为
**10s**
本软件在使用过程中会滤过已经选的课以及满的课，一旦出现空课，会立即弹窗提示。

欢迎各位参与本软件的测试工作
